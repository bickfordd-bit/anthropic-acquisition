# Anthropic Acquisition Package

Interactive web experience for presenting Bickford acquisition opportunity to Anthropic leadership.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com
3. Import this repository
4. Deploy (no configuration needed)

## Deploy to Netlify

1. Go to https://app.netlify.com/drop
2. Drag `index.html` into the drop zone
3. Get instant URL

## Customize

Edit `index.html` to update:

- Phone number (search for `tel:+1234567890`)
- Email address (search for `derek@bickfordtechnologies.com`)
